## 🔧 Tips for Better Forms

Keep these tips in mind while you build:

- Use short field labels
- Ask one thing at a time
- Keep required fields to a minimum
- Group related questions together
- Use logic to hide extra questions
- Test the form before sharing it
- Check how it looks on a small screen

If a form feels long, trim it. If a question feels unclear, rewrite it in simple words.
